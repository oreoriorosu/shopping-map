# ShoppingMap 機能一覧

最終更新: 2026-04-20

---

## マップ操作

| 機能 | 操作方法 |
|------|---------|
| PDFマップ追加 | ヘッダーの ↑ アイコンをタップ |
| マップ切替 | ヘッダーのドロップダウンから選択 |
| ズーム/パン | ピンチイン/アウト、スワイプ |
| ズームイン・アウト・リセット | マップ上部のボタン |
| 複数ページ切替 | マップ上部の < / > ボタン（複数ページPDFのみ） |
| ズーム位置の永続化 | マップごとに最後の表示位置をローカル保存。リロード後も復元される |
| 優先度フィルター | マップ上部の A/B/C/D ボタンをタップして絞り込み（複数選択可） |
| 完了済みスポット非表示 | マップ上部の「済み非表示」ボタン |

## スポット（サークル）

| 機能 | 操作方法 |
|------|---------|
| スポット追加 | マップ表示中に右下の **＋** FAB → 情報入力 → マップをタップしてピン配置 |
| ピン配置プレビュー | ピン配置モード中に長押しすると「ここに配置」プレビューが表示される |
| スポット編集 | リストのスポット名右のペンアイコンをタップ |
| スポット削除 | 編集モーダル下部の「このサークルを削除」 |
| ピン移動 | マップ上部のペンシルボタンでピン編集モードに入り、ピンを長押し＆ドラッグ |
| ピンタップ → ポップアップ | マップ上のピンをタップすると商品リストの概要ポップアップが表示される |
| ポップアップ → リストへジャンプ | ポップアップの店名をタップするとリストタブのスポットにスクロール移動 |
| スポット完了チェック | 商品が登録されていないスポットはリスト右端のチェックボタンで完了にできる |

**スポット入力項目**

| 項目 | 内容 |
|------|------|
| 場所 | ブース番号（例: さ-10） |
| サークル名 | 省略すると場所名が使われる |
| 優先度 | A / B / C / D（任意） |
| 推し | キャラ名など（任意） |
| ジャンル | 漫画・イラストなど（任意） |
| ピンカラー | 8色から選択 |
| お品書き画像 | 端末から画像を選択（任意） |

## 買い物リスト

| 機能 | 操作方法 |
|------|---------|
| 全マップ横断表示 | リストタブでは全マップのスポットをまとめて表示 |
| 商品追加 | スポット行下部の「＋ 商品を追加」をタップ。名前＋金額（任意）を入力 |
| 購入済みチェック | 商品行左の丸ボタン |
| 売切マーク | 商品行右の「売切」ボタン |
| 価格集計 | ヘッダーに合計金額と購入済み金額を表示（金額入力済みの場合のみ） |
| 進捗バー | ヘッダーの緑/赤バー（緑: 購入済み、赤: 売切） |
| 未購入フィルター | ヘッダーの「未購入」ボタンで未チェックスポットのみ表示 |
| スポット並び替え | ヘッダーの「並び替え」ボタン → ドラッグ or 「名前順」「優先度順」ボタン |
| スポット折りたたみ | スポット行左の ∨ / ＞ ボタン |
| マップのピンへ移動 | スポット行右のピンアイコンをタップ → マップタブに切り替わりピンが中央に表示 |

## その他

| 機能 | 内容 |
|------|------|
| お品書き画像表示 | リストのスポット行右サムネイル or マップのポップアップからフルスクリーン表示 |
| バックキー / スワイプバック対応 | モーダル → ピン配置キャンセル → リストタブ の順に状態を閉じる |
| 複数マップ対応 | マップをまたいでスポット管理。リストは全マップ横断で表示 |
| データ保存先 | ブラウザの IndexedDB（ローカル保存、バックエンドなし） |
